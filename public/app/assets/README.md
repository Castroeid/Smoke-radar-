# Smoke Radar App Visual Assets

The app supports optional campaign-style image assets in this folder.

Recommended files:
- `app-bg-smoke.jpg`
- `hero-steak.jpg`
- `butcher-map-bg.jpg`
- `recipe-card-bg.jpg`
- `hot-trends-bg.jpg`

If these files are not present, the app still renders with layered dark gradients, smoke-like overlays, ember highlights, and CSS-generated map/radar accents.
